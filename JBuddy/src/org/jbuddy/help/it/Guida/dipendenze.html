Gestione Dipendenze

Alcune volte il codice Java deve far riferimento a librerie esterne (non jre) per poter svolgere il proprio compito.

Esempi semplici possono essere:

  • Connessioni alle basi di dati JDBC
  • JavaHelp
  • Java2D
  • molte altre...

La possibilità di usare librerie esterne con JBuddy è utile per provarle senza fare troppa "infrastruttura"

Per poterle usare è necessaria una semplice configurazione


Configurazione di librerie esterne

Aprire la finestra di gestione delle dipendenze dal menu Strumenti scelta Gestione Dipendenze

Si presenterà una finestra come quella sottostante:


Da questa finestra è possibile aggiungere riferimenti esterni a file Jar e a directory che contengono delle classi Java
