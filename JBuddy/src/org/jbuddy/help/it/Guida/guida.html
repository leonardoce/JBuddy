Cosa è JBuddy


Java è un linguaggio dotato di una libreria di classi molto potente e di molti pacchetti disponibili gratuitamente che permettono di soddisfare anche le esigenze più particolari.

Ci sono anche molti ottimi ambienti di sviluppo, sia in ambito open-source che in ambito commerciale. Molti di questi ambienti, però, sono molto complessi e quindi diventano difficili da usare soltanto per scopi didattici oppure solamente per sperimentare una libraria esterna.

JBuddy è nato per risolvere questi problemi perchè:

JBuddy è un programma nato per poter sperimentare facilmente il linguaggio Java usando un ambiente il più possibile semplice ed immediato da apprendere.
